## 指针样式

- [x] Default 默认
- [x] Link 点击链接
- [x] Unavailable 不可用
- [x] Zoomin 放大
- [x] Zoomout 缩小
- [x] Handwriting 手写
- [ ] IBeam 选定文本
- [ ] SizeNS 上下拉伸
- [ ] SizeEW 左右拉伸
- [ ] SizeNESW 左下右上
- [ ] SizeNWSE 左上右下
- [ ] SizeAll 全方位
- [ ] Copy 复制
- [ ] Cut 剪贴
- [ ] Help 帮助事项
- [ ] Wait 加载中

## 设计参数

- Size: 
  - light 25×25px
  - normal 37×37px
  - blod 45×45px
- 透明度规范: 90/80/74/50%
- 线条规范: 9/7/4px
- color: #2c2c2c #fff #fc5555 #f8a4a4
